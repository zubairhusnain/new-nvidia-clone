    Use Cases

    How NVIDIA AI Workbench Is Being Used

    See some examples of NVIDIA AI Workbench accelerating developer workflows in the real world.

    1. AI Chatbot
    2. Model Fine-Tuning
    3. Custom Image Generation

    AI Chatbot With RAG

    Easily create, use, and distribute RAG-powered AI chatbots with AI Workbench. Check out:

    • Hybrid-RAG for rapid prototyping: Run embedding locally and inference locally or in the cloud with NVIDIA NIM™ or an NVIDIA API.
    • Agentic RAG for improved response quality: Use AI agents for dynamic information retrieval across documents and web search. Run inference in the cloud with NVIDIA NIM or an NVIDIA API. 
    • NIM-Anywhere to get started today: Implement native RAG and build your own AI-powered tools with NVIDIA NIM.
    Create Your Own AI Chatbots

    Fine-Tuning LLMs At Any Scale

    Customize LLMs and tailor to specific domains. Choose from a variety of models, frameworks, and fine-tuning methods,and fine-tune with custom datasets to increase domain-specific knowledge. 

    Simplify environment setup and container configuration across local and remote deployments with AI Workbench. 

    Get started with AI Workbench example projects available on NVIDIA’s GitHub to customize Llama 3, Mistral, Phi 3, and more.

    View All Fine-Tuning AI Workbench Projects

    Custom Image Generation With Stable Diffusion XL (SDXL)

    AI Workbench simplifies local deployment of Stable Diffusion XL models. It manages environment setup, model acquisition, and container configuration. AI Workbench also supports remote workstations for running Jupyter Notebooks and allows fine-tuning of models for specific subjects, enhancing image generation results.

    Try SDXL AI Workbench Project
